# Hello Elementor Theme

A lightweight, fast, and minimalist WordPress starter theme built specifically to work seamlessly with the **Elementor Page Builder**.  
Designed as a clean foundation for designers, developers, and Elementor users who want full control over their site layout.

---

## Features

- Ultra-lightweight and fast  
- Fully compatible with Elementor  
- Clean markup and minimal styling  
- Flexible and easy to customize  
- Ideal for building sites with a child theme or custom CSS  
- SEO-ready, mobile-responsive, and developer-friendly  

---

## Requirements

- **WordPress:** 6.0 or above  
- **PHP:** 7.4 or above  

---

## Description

Hello Elementor is an open-source WordPress theme built to give users a blank, optimized canvas for Elementor.
